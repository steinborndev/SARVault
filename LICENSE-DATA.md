# Data License & Attribution

## ChEMBL

This project uses bioactivity data from **ChEMBL**, maintained by the European
Bioinformatics Institute (EMBL-EBI).

- Source: https://www.ebi.ac.uk/chembl/
- Release: pinned via `config/target_set.yml` (`chembl_version`)
- License: Creative Commons Attribution-ShareAlike

ChEMBL data is redistributed here only as small derived fixtures for testing.
No bulk ChEMBL dataset is committed to this repository. Any published derivative
of ChEMBL data carries the ShareAlike obligation.
